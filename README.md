# Система Алгебра

WPF-приложение для изучения школьного курса алгебры.

## Цель проекта
Разработка информационной системы для изучения тем алгебры, решения квадратных уравнений и просмотра математических таблиц.

## Функциональность
- Просмотр и добавление тем (алгебраическая дробь, упрощение выражений, квадратный корень)
- Решение квадратных уравнений (дискриминант / теорема Виета)
- Таблицы квадратов, кубов, факториалов
- Поиск по таблицам
- Обработка ошибок ввода

## Технологии
- C# / WPF
- PostgreSQL
- Npgsql
- MSTest

## База данных
Таблицы: `Topics`, `NumberSquares`, `NumberCubes`, `NumberFactorials`, `EquationHistory`

## Установка и запуск

### Требования
- Visual Studio 2022
- PostgreSQL 15+
- .NET 8.0/9.0

### Шаги
1. Клонировать репозиторий
2. Создать базу данных `AlgebraDB` в pgAdmin 4
3. Выполнить SQL-скрипт из файла `database.sql`
4. Открыть проект в Visual Studio
5. Запустить (F5)

## Тестирование
- Фреймворк: MSTest
- Количество тестов: 15
- Запуск: Test Explorer в Visual Studio
